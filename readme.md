🎡Интерактивное веб-приложение для проведения розыгрышей в прямом эфире. Позволяет вводить список участников, крутить анимированное колесо и сохранять историю победителей.

## ✨ Особенности
*   **Динамическое колесо**: количество секторов меняется автоматически в зависимости от списка имен.
*   **Эффектная анимация**: плавное замедление колеса и всплывающее окно при победе.
*   **Без зависимостей**: проект написан на чистом JavaScript (Vanilla JS) без использования тяжелых библиотек.

## 🚀 Быстрый старт

Проект включает в себя **Makefile** для удобного управления через терминал.

### Основные команды (требуется Node.js):


| Команда | Описание |
| :--- | :--- |
| `make run` | Запустить сервер через `npx serve` на порту 3000 |
| `make start` | Запустить сервер и открыть браузер по умолчанию |
| 'make help' | Список доступных команд |
| 'make open' | Открытие браузера по умолчанию |

🏎 Скорость и время вращения
В файле index.js внутри обработчика spinBtn.addEventListener можно настроить:

    spinAngleStart: Начальная скорость (чем выше число, тем быстрее крутится).
