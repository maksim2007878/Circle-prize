### 🌟 О проекте
Профессиональное решение для проведения интерактивных розыгрышей. Это не просто статичная страница, а полноценное клиент-серверное приложение с автоматическим сохранением результатов в локальную базу данных.

---

### 🚀 Функциональные возможности

*   🎯 **Smart Canvas**: Динамическая отрисовка секторов под любое количество участников.
*   💾 **Persistent History**: Автоматическая запись каждой победы в `history.json` (Имя, Дата, Время).
*   🔄 **Data Sync**: История не пропадает при обновлении страницы — сервер подгружает её мгновенно.
*   🧹 **Admin Tools**: Встроенная функция полной очистки истории через интерфейс.

### 📦 Быстрый старт

1. **Подготовка окружения** (необходим Node.js):
   ```
   npm install express
   ```
2. **Запуск через Makefile:**
   ```
   make start
   ```
3. **Ручной запуск:**
   ```
   node server.js
   ```
После чего перейдите по адресу: http://localhost:5000

⌨️ Доступные команды

    make run — Запуск только серверной части.
    make open — Открытие текущей ссылки в браузере по умолчанию.
    make start — Полный цикл: Сервер + Браузер.

⚙️ Кастомизация
Для изменения визуальных настроек используйте переменные в коде:

    Цвета секторов: массив colors в файле index.js.
    Порт сервера: переменная PORT в server.js, index.js и Makefile.
    Фон: блок body в style.css.
